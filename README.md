# Kill Clone Tabs

**Kill Clone Tabs** is a powerful Chrome extension designed to help you manage your browser tabs efficiently. It allows you to find and close duplicate tabs, sort and group them, and provides useful tools for highlighting text and copying tab information.

## Key Features

### üßπ Tab Management
*   **Kill Clones**: Instantly find and close duplicate tabs to free up memory and declutter your browser.
*   **Sort Tabs**: Organize your tabs by **URL** or **Title** with a single click.
*   **Group Tabs**: Automatically group tabs by their domain (e.g., all google.com tabs together).
*   **Undo**: Made a mistake? Use the Undo button to revert your last Sort, Group, or Kill action.

### üõ†Ô∏è Bulk Actions
*   **Pin/Unpin All**: Quickly pin or unpin all tabs in the current window.
*   **Mute/Unmute All**: Mute or unmute audio in all tabs at once.
*   **Copy Tabs**:
    *   **Copy All**: Copy the Title and URL of *all* open tabs to your clipboard.
    *   **Copy Selected**: Copy the Title and URL of currently *selected* tabs.
*   **Export**: Export your list of open tabs to a Markdown file.

*   **Customizable**:

    *   Toggle **Case Sensitivity**.
    *   Match **Whole Words** only.
    *   Set a **Minimum Character Length** for highlighting.

## Installation

1.  Clone this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** and select the extension directory.

## Usage

1.  Click the **Kill Clones** icon in your browser toolbar to open the popup.
2.  **To Close Duplicates**: Click the large "Knife" icon.
3.  **To Sort/Group**: Use the buttons in the "Organize" section.
4.  **To Copy/Pin/Mute**: Use the icons in the "Manage" section.
5.  **To Configure**: Click the "Settings" (Gear) icon to access options for Whitelist, Highlighting, and more.

## Privacy

This extension runs locally on your device. It does not collect or transmit any personal data or browsing history.
